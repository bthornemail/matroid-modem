# RFC-0012

# Civic Constitutional Canon Compiler v1.0

## Formal Grammar for Founding Documents → Civic Triples

Status: Draft
Depends on: RFC-0003

---

# 1. Purpose

Defines formal grammar for compiling civic/legal texts into canonical SPO triples.

---

# 2. Lexical Layer

Tokens:

```id="cc1"
NOUN
VERB
MODAL
NEGATION
ARTICLE
CONJUNCTION
CLAUSE
```

---

# 3. BNF Grammar

```id="cc2"
<sentence> ::= <clause> | <clause> ";" <clause>

<clause> ::= <subject_phrase> <predicate_phrase> <object_phrase>

<subject_phrase> ::= <noun_phrase>

<predicate_phrase> ::= <modal>? <verb_phrase>

<object_phrase> ::= <noun_phrase> | <clause>

<noun_phrase> ::= NOUN | NOUN <modifier>

<modifier> ::= "of" NOUN | ADJECTIVE NOUN
```

---

# 4. Modal Interpretation

| Modal     | Mode        |
| --------- | ----------- |
| shall     | mandatory   |
| may       | permissive  |
| must      | binding     |
| shall not | prohibition |

---

# 5. Negation Handling

Negation attaches to predicate:

```id="cc3"
shall not make → prohibit(make)
```

---

# 6. Triple Extraction

Given:

> "Congress shall make no law"

Compiler produces:

```json id="cc4"
{
  "subject": "Congress",
  "predicate": "prohibit_make_law",
  "object": "law",
  "modal": "mandatory",
  "polarity": "negative"
}
```

---

# 7. Canonicalization Rules

* Lowercase
* Lemmatize verbs
* Strip punctuation
* Normalize plurals

---

# 8. Synset Anchoring

Each token mapped to WordNet synset deterministically.

Example:

```id="cc5"
"law" → law.n.01
"make" → create.v.01
```

---

# 9. Rumsfeldian Classification

Compiler annotates:

| Mode | Condition           |
| ---- | ------------------- |
| KK   | explicit rule       |
| KU   | delegated authority |
| UK   | ambiguous reference |
| UU   | implied silence     |

---

# 10. Output Record

```json id="cc6"
{
  "type":"civic_triple",
  "epistemic_mode":"KK"
}
```

---

# 11. Determinism

Compiler MUST produce identical triples for identical input text.
