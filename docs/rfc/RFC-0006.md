# RFC-0006

# SAB Projection Canonical Geometry v1.0

## Deterministic 16×16 Semantic Projection

Status: Draft
Depends on: RFC-0001

---

# 1. Purpose

Defines canonical mapping from semantic coordinates to the 16×16 Shared Array Basis (SAB).

This ensures identical projection across implementations.

---

# 2. Grid Layout

16×16 grid indexed:

```
row ∈ [0..15]
col ∈ [0..15]
```

---

# 3. Row Regions

| Rows  | Region               |
| ----- | -------------------- |
| 0–11  | Golden simplex plane |
| 12–15 | Meta / UU layer      |

---

# 4. Column Mapping

Given w ∈ [0,1]:

```
col = floor(w * 16)
if col == 16 → col = 15
```

---

# 5. Face Mapping to Rows

Define:

| Face      | Row Offset |
| --------- | ---------- |
| subject   | 0          |
| predicate | 1          |
| object    | 2          |
| centroid  | 3          |

For meta rows:

```
row = 12 + row_offset
```

Example:

* predicate + w=0.125
* col = floor(0.125 * 16) = 2
* row = 12 + 1 = 13

Lights (13,2)

---

# 6. Quadrant Color Encoding

| Mode | Color  |
| ---- | ------ |
| KK   | green  |
| KU   | yellow |
| UK   | orange |
| UU   | blue   |

Mapping derived from epistemic classification.

---

# 7. synset_call Projection

On synset_call:

* Light cell (row, col) KU
* Duration: CALL_DURATION_MS

---

# 8. synset_reply Projection

If faces_passed ≥ quorum:

* Pulse KK
  Else:
* Pulse UK

---

# 9. commit Projection

Flash centroid row (row 15) at col derived from w=0.5.

---

# 10. Canonical Projection Invariant

Given identical:

* independent event set
* resolution output

Projection MUST produce identical 16×16 state matrix.

Matrix must be serializable:

```json
{
  "sab_state": [[0,0,...],[...],...]
}
```
